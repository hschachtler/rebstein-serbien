GitHub Verbindung
Repository & Token einrichten (einmalig)
Einmalige Einrichtung — Repo-URL reicht zum Lesen, Token nur zum Bearbeiten:
1. Trage die Repo-URL ein (z.B. max/bikepacking-tour)
2. Für Admin-Zugriff (Schreiben): Token unter github.com → Settings → Developer Settings → Personal Access Tokens erstellen
3. Token mit Scope repo (Classic) oder Contents: Read & Write (Fine-grained)
📸
Foto-Tagebuch einrichten
Cloudinary – kostenlos, 25GB Speicher
Einmalige Einrichtung:
1. cloudinary.com ↗ → kostenlos registrieren
2. Dashboard → Cloud Name notieren
3. Settings → Upload → Add upload preset → Signing Mode: Unsigned → Save
1
Tour anlegen oder wählen
Jede Tour hat eigene Route, POIs und Einstellungen
Lege zuerst deine Tour an — alle folgenden Schritte (GPX, POIs, Tracking) werden dieser Tour zugeordnet.

Du kannst mehrere Touren verwalten, aber immer nur eine ist öffentlich sichtbar.
➕ Neue Tour erstellen
2
GPX von Komoot importieren
Route analysieren & bei Bedarf splitten
So gehts:
1. Öffne deine Tour auf komoot.de
2. Klicke auf … → Tour exportieren → GPX
3. Lade die Datei hier hoch — die App prüft automatisch ob ein Split nötig ist
📁 GPX-Datei hierher ziehen
oder tippen zum Auswählen
3
onroutemap.de — POIs hinzufügen
GPX importieren, POIs mit Details ergänzen, exportieren
So gehts (je GPX-Teil):
1. Öffne onroutemap.de ↗
2. GPX hochladen → POIs suchen und hinzufügen
3. Die App importiert POIs mit spezifischem Namen und sinnvollem Typ automatisch
4. GPX wieder exportieren
5. Wiederhole für alle Teile
4
Angereicherte GPX importieren
POIs aus onroutemap laden — mehrere GPX-Dateien möglich
Lade alle von onroutemap exportierten GPX-Dateien hoch.
Die App filtert automatisch nach Qualität und zeigt was importiert wird.
✅ Kriterien: spezifischer Name · sinnvoller Typ · max. 500m von der Route
📥 Eine oder mehrere GPX-Dateien hierher ziehen
oder tippen — Mehrfachauswahl möglich
5
POI-Liste prüfen & speichern
Eigene POIs ergänzen, dann in GitHub sichern
Fast fertig!
• Prüfe die POI-Liste und ergänze eigene Einträge
• Speichere alles in GitHub – dann sind Tour und POIs auf allen Geräten sichtbar
Speichert Route (GPX-Trackpunkte) und alle POIs
6
Tour veröffentlichen & Live-Tracking
Tour für alle sichtbar + GPS-Tracking starten
Wenn du losfährst:
1. Öffne die App auf deinem Handy
2. Melde dich als Admin an (🔐 Login)
3. Gehe zur Karte und tippe Admin → Tracking aktiv
4. Deine Position wird alle 15 Sek. in GitHub gespeichert

Alle mit dem Link sehen deinen Standort live auf der Karte! 🎉
